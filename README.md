# Customer Support Ticket System

A simple Python-based command-line application that simulates a real-world customer support workflow, allowing users to create, track, and manage support tickets.

---

## 🚀 Features

* Create new support tickets
* View all existing tickets
* Update ticket status (Open, In Progress, Closed)
* Persistent storage using JSON
* Simple CLI interface for interaction 

---

## 🧠 How It Works

Users interact with a command-line menu to:

1. Submit a support request
2. View all submitted tickets
3. Update the status of a ticket

All tickets are stored in a local `tickets.json` file.

---

## ⚙️ Project Structure

```text
main.py              # CLI interface
ticket_manager.py    # Ticket logic and operations
tickets.json         # Data storage
```

---

## ▶️ How to Run

1. Clone the repository:

```bash
git clone https://github.com/YOUR-USERNAME/customer-support-ticket-system.git
```

2. Navigate into the folder:

```bash
cd customer-support-ticket-system
```

3. Run the application:

```bash
python main.py
```
